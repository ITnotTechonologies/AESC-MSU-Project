# Практическое задание №2
Сайт для доставки в СУНЦ
1. Целевая аудитория: Ученики СУНЦ, воспитатели, преподаватели.
2. От кого? Ученики 11Е СУНЦ МГУ.
3. Основная идея: Сайт содержит страницу с выбором курьера, который принесет ваш
заказ в любую точку на территории СУНЦа . На сайте будет чат для связи с курьером,
страница с выбором товара. Будет отдельная категория доставки для товаров из
OZON’a и ЯНДЕКС маркета. Будет версия и для телефонов и для компьютеров.
# Структура сайта:
1. Главная страница
2. Профиль пользователя
3. Каталог товаров
4. Страница оформления заказа
5. Страница отслеживания состояния заказа
6. Чат с курьером и поддержкой
7. История заказов
8. Страница 404
9. История создания проекта и страница с контактными данными
