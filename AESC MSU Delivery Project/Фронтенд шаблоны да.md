Ниже — шаблоны, ориентированные на MVP из ТЗ: страницы главной, каталога, профиля, оформления заказа, отслеживания, чата и 404; стек также соответствует требованиям — FastAPI, Jinja2, TailwindCSS и минимум сложного JS.

## Компоненты

1. `base.html` — общий layout, навигация, подвал, подключение TailwindCSS и общий JS.
    
2. `partials/_flash.html` — сообщения об успехе/ошибках.
    
3. `partials/_status_badge.html` — цветной бейдж статуса заказа.
    
4. `partials/_product_card.html` — карточка товара.
    
5. `partials/_order_card.html` — карточка заказа в истории.
    
6. `home.html` — hero, преимущества, популярные товары, курьеры.
    
7. `catalog.html` — поиск, фильтры, сортировка, сетка товаров.
    
8. `profile.html` — данные пользователя, роль, последние заказы.
    
9. `order_create.html` — оформление заказа, выбор курьера и точки доставки.
    
10. `order_track.html` — статус заказа, история статусов, автообновление.
    
11. `chat.html` — чат с курьером.
    
12. `404.html` — дружелюбная страница ошибки.

<script>
  const orderId = document.querySelector('meta[name="order-id"]')?.content;
  async function refreshOrder() {
    if (!orderId) return;
    try {
      const response = await fetch(`/orders/${orderId}/track-fragment`, { headers: { 'X-Requested-With': 'fetch' }});
      if (!response.ok) return;
      const data = await response.json();
      const badge = document.querySelector('[data-order-status]');
      const lastUpdated = document.getElementById('last-updated');
      if (badge && data.status_label) badge.textContent = data.status_label;
      if (lastUpdated && data.last_updated) lastUpdated.textContent = data.last_updated;
    } catch (e) {}
  }
  setInterval(refreshOrder, 7000);
</script>
